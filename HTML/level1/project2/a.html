what is html?

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

what is Structure?

The two primary structural components in HTML are the body and the head. Both of these are used by all web browsers to render the structure and settings of a webpage.

what are the types of css in html?

There are three types of CSS: external, internal, and inline. External CSS commands are written in a separate file from the main HTML page. These are thenf linked to each web page by a command in the HTML file.
